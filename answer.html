Q & A

A.What is DOM and what is the purpose of DOM?

Ans:-Dom stands for Document Object Model. Basically it helps to present the interface of a document.With the help of DOM browser shows us webpages.

B.How will you select HTML elements using DOM? Name the DOM methods.

Ans:-We can select HTML elements by some methods like getElementById(),getElementsByClassName(),getElementsByTagName(),querySelector(),querySelectorAll().

C.What is event bubble?

Ans:-It is part of Document Object Model. When an element hit's an event it happened.

D.What is a callback function and why will you use it?

Ans:-When a function passed itself as an argument into another function and does an action or return action. Basically we will use it when we are doing same functional work of some given task.
